- Go 端导出给 Wails 调用的方法（如 `GetTunnels`, `StartTunnel`）直接定义在 `App` 结构体上，并返回结构化数据或标准 error。
- 前端 API 层（`frontend/src/api/app.ts`）统一封装 Wails 运行时调用，并提供 `createPreviewBinding` 以支持非 Wails 环境下的浏览器预览。
- 状态管理采用 Pinia Composition API 风格（`defineStore` 返回 setup 函数），将状态、计算属性与异步动作集中定义。
- 后端配置存储模块（`internal/config`）使用 `database/sql` 原生接口配合占位符查询，确保对 SQLite 的安全访问。
