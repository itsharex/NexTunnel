- 采用 Wails v2 框架构建 Go (后端) + Vue 3 (前端) 的混合架构，通过 `wails.json` 配置构建流程。
- 后端入口 `main.go` 初始化 `App` 结构体并绑定至 Wails 运行时，负责生命周期管理。
- `internal/tunnel` 模块作为核心业务层，管理隧道配置、连接状态及与 Relay 服务器的控制信令交互。
- `internal/p2p` 和 `internal/quic` 模块实现底层网络能力，包括 STUN NAT 检测、ICE 候选交换、UDP 打洞及 WireGuard/QUIC 数据传输。
- 前端位于 `frontend/` 目录，使用 Vite + TypeScript + Vue 3，通过 `wailsjs` 自动生成的桥接代码调用后端 Go 方法。
- 数据持久化通过 `internal/config` 模块基于 SQLite (`modernc.org/sqlite`) 实现，存储隧道配置与应用设置。
