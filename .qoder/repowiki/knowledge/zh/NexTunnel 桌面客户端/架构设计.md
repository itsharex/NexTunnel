- **混合架构**：采用 Wails v2 实现 Go 后端与 Vue 3 前端的深度集成。后端 `main.go` 作为入口，通过 `embed.FS` 注入前端静态资源，并将 `App` 结构体绑定至运行时以暴露 RPC 接口。
- **后端分层**：`app.go` 充当应用控制器，协调 `internal/tunnel`（隧道生命周期管理）、`internal/config`（基于 SQLite 的配置持久化）以及 `internal/p2p`（P2P 连接引擎）。
- **前端技术栈**：使用 Vue 3 + TypeScript + Vite 构建 UI，通过 Pinia (`stores/tunnel.ts`) 管理全局状态，利用 Naive UI 组件库实现深色主题界面。
- **通信机制**：前端通过 `wailsjs` 自动生成的代理调用后端方法，后端通过 `slog` 记录日志并管理隧道连接状态。
