- **混合架构**：采用 Wails 框架实现 Go 后端与 Vue3 前端的深度集成。Go 层负责核心网络逻辑（隧道管理、P2P 引擎、中继连接），前端负责 UI 交互与状态展示。
- **后端分层**：`main.go` 为入口，`app.go` 作为 Wails 绑定层暴露 API；`internal/tunnel` 封装隧道生命周期管理，`internal/p2p` 实现 NAT 穿透与 WireGuard 隧道建立，`internal/config` 基于 SQLite 持久化配置。
- **前端结构**：使用 Vue 3 + TypeScript + Vite，通过 Pinia (`stores/tunnel.ts`) 管理全局状态，调用自动生成的 `wailsjs` 桥接代码与后端通信。
- **依赖方向**：前端依赖后端暴露的 API，后端内部模块通过接口解耦（如 `P2PEngine` 接口），核心网络库复用 `github.com/nextunnel/pkg`。
