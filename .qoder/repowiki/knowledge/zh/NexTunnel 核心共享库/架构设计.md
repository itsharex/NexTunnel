- 采用扁平化子包结构，各功能域（protocol, crypto, ipam, audit, tlsutil, types）相互独立，无内部循环依赖。
- `protocol` 包定义了基于 JSON 负载的二进制帧协议（5字节头+负载），并通过 `codec.go` 提供线程安全的连接封装 `Conn`。
- `types` 包集中管理跨组件共享的数据结构，如隧道配置、代理状态及智能调度指标。
- `audit` 包通过 `AuditLogger` 接口抽象日志存储，支持 SQLite（带索引查询）和 JSON 文件两种实现。
- `crypto` 与 `tlsutil` 分别处理底层密钥生成（Curve25519/PSK）和上层 mTLS 证书链管理，支撑安全通信。
