- 所有服务入口采用统一的信号驱动优雅停机模式：监听 `SIGINT/SIGTERM`，执行 `Stop/Shutdown` 并记录启停日志。
- HTTP API 统一使用 `writeJSON/writeAPIError`（或 `writeSuccess/writeError`）封装响应，保持 `Content-Type: application/json` 与一致的错误结构。
- 共享配置通过 `Default*Config()` 提供默认值，并结合 `flag` 解析覆盖，避免硬编码启动参数。
- 并发访问保护采用读写锁（`sync.RWMutex`）或原子变量（`atomic.Value/Int32`）隔离状态读写，避免全局锁竞争。
- 关键路径鉴权使用 `crypto/subtle.ConstantTimeCompare` 进行常量时间比较，防止时序侧信道泄露。
- 内部包通过 `doc.go` 声明包职责与边界，形成稳定的内部模块文档化约定。
