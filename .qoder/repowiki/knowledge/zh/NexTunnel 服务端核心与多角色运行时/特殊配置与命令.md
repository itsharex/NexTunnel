- 多二进制构建：分别编译 `cmd/control-plane`、`cmd/relay`、`cmd/nat-detector`（默认 `go build ./cmd/...`）。
- 控制面启动参数：`-listen`、`-api-token`、`-store-path`（空则内存），见 `cmd/control-plane/main.go`。
- 中继启动参数：通过 `relay.ParseFlags` 解析，支持 `-stats-interval` 周期性统计日志，见 `cmd/relay/main.go`。
- NAT 检测启动参数：`-primary-addr`、`-alt-addr`、`-port`、`-realm`，见 `cmd/nat-detector/main.go`。
- 前端开发：`web/` 使用 Vite + Vue + TypeScript（`vite.config.ts`、`package.json`），需 `npm install` 后启动 dev server。
