- **多入口架构**：通过 `cmd/` 下的 `control-plane`（HTTP API 管理）、`relay`（TCP/QUIC 数据转发）和 `nat-detector` 实现功能解耦。
- **内部模块化设计**：`internal/` 包含核心逻辑，如 `controlplane`（节点注册与密钥交换）、`relay`（基于 `quic-go` 的多路复用传输）、`anycast`（基于 GeoIP 的就近路由）以及 `sdwan`（流量分类与 QoS 策略）。
- **硬件加速层**：`ebpf` 模块提供内核态数据包转发能力，并支持回退至用户态转发。
- **可视化与管理**：`dashboard` 模块提供基于 Vue3 前端和 Go 后端的监控告警界面。
- **依赖方向**：各业务模块依赖 `pkg` 共享协议定义，控制面负责状态维护，数据面负责高并发连接处理。
