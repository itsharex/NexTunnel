- 多入口架构：通过 `cmd/` 下的四个独立二进制（control-plane, dashboard, relay, nat-detector）分别承载不同职责。
- 内部模块化：`internal/` 包含 controlplane（节点注册、ACL、IPAM）、relay（QUIC/TCP 流量代理）、dashboard（Web UI 后端、RBAC）、edge（节点健康探测）及 sdwan/anycast（路由与策略）等子包。
- 存储抽象：控制平面与仪表板均定义 `Store` 接口，支持 SQLite (`modernc.org/sqlite`) 持久化或内存模式，实现业务逻辑与存储解耦。
- 安全通信：全链路支持 mTLS（基于 `pkg/tlsutil`），控制平面支持 Bearer Token 与客户端证书双重认证，中继服务支持共享 Token 鉴权。
