- **多入口架构**：通过 `cmd/` 下的四个独立入口（control-plane, dashboard, relay, nat-detector）启动不同职责的微服务。
- **模块化内部设计**：`internal/` 目录按功能垂直切分，包含控制平面（节点注册/ACL）、中继服务（QUIC/TCP 转发）、边缘计算（节点健康监控）及 eBPF 内核态转发模块。
- **存储抽象层**：控制平面与仪表盘均采用 Store 接口模式，支持 SQLite 持久化与内存模式的无缝切换（如 `store_sqlite.go` 与 `store_memory.go`）。
- **前端集成**：内置基于 Vue 3 + Vite 的 Web 管理界面（`web/`），由 Dashboard 服务在运行时托管静态资源。
