# 桌面端能力总览

v0.4.1-alpha 聚焦桌面端可观测性、配置效率、CLI 协同管理和真实 TUN 生产预检。

## 实时流量反馈

总览页会按刷新周期保存流量快照，并通过 `uPlot` Canvas 时序图显示上传、下载增量曲线。图表不使用动画，页面微交互仅使用 `transform` 和 `opacity`，并支持 `prefers-reduced-motion` 降级。

## 精简总览页

总览页只保留连接控制、核心流量指标、实时流量图和最近日志。端口管理与隧道管理移动到隧道页，降低首页信息密度。

## 连接类型

隧道列表新增连接类型字段：

| 类型 | 含义 |
| --- | --- |
| P2P 直连 | P2P 引擎明确进入 connected 状态 |
| 中继 | 隧道运行中，但当前数据面通过 Relay |
| 待机 | 隧道未运行或未注册 |

## 设置界面

设置页改为分类导航，包含连接、网络、端口、安全、外观和关于。这个结构更接近现代桌面开发工具，适合后续持续扩展。

## 运行日志

桌面端新增持久化运行日志，覆盖连接、隧道、设置、虚拟网络、NAT 探测、端口扫描和运行错误。首页展示最近日志，完整日志页支持级别和分类筛选、刷新和清空。

## 图标导航

左侧导航使用 Lucide 图标替代数字编号，并提供滑动指示条和轻量页面切换动画。

## 真实 TUN 生产预检

网络页会展示当前平台的生产数据面模式、真实 TUN 是否就绪、阻塞项和降级项。Windows 会检测管理员权限和 `wintun.dll`，macOS 会提示 sudo/root 权限要求，Linux 会检测 `/dev/net/tun` 与 `CAP_NET_ADMIN`。这些检查只用于明确生产前置条件，不会把用户态 `netTun` 回退伪装成生产可用。
